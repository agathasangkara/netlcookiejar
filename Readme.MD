# Apa itu Netflix Device Manager?

Netflix Device Manager Adalah?

Sebuah tool yang dapat membantu Anda mengelola akun Netflix dengan mudah.

Menu yang Tersedia didalam bot sebagai berikut:

Tambah akun:
Tambahkan akun Netflix baru menggunakan cookie. Cookie digunakan untuk login tanpa password.

Hapus akun
Hapus akun Netflix yang sudah tidak digunakan dari daftar Anda.

Cek Akun
Lihat semua akun Netflix yang tersimpan, informasi profil, device aktif, dan kelola akun Anda (pindah profil, hapus device, dll).

Pada intinya Netflix Device manager adalah tool otomatis yang digunakan untuk mengelola akun netflix anda, dari profil , device etc
